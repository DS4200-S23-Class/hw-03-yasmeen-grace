Channels

  • Channel 1:color
    • Represents attribute of type quantitative electoral votes
    • This is a good mapping because color makes the map stand out and helps the user understand that there is data trying to be represented. Otherwise, it would just look like the map of the United States.
  • Channel 2: shape
    • Represents attribute of quantitative votes per state
    • This is a good channel because it outlines specific states to show contrast and comparison. The small rectangles at the bottom help break up the individual states into more categories to specify how the data is distrbuted within the state.

Colormaps

  • Categorical
    • Represents attribute of type political party
    • This is a good choice because it helps distinguish betweem the two candidates and states to reveal which are more Democratic or Republican.

Channels

  • Channel 1:color
    • Represents attribute of type quantitative angle and speed
    • This is a good mapping because it helps the user distinguish higher and lower scoring balls.
  • Channel 2: position(horizontal and vertical)
    • Represents categorical attribute of type of ball hit i.e ground, fly, etc.
    • This is a good channel because it helps distinguish the types of balls hit and the position at which they land. It also shows the differening exit velocities that they were hit.

Colormaps

  • Divergent
    • Represents attribute of type quantitative scoring value.
    • This is an okay mapping. The colormap can be divergent, but it would be easier to distinguish the scoring value differences with colors that are more distinct. For example, maybe using orange and blue instead. A sequential color map could also work with the darker shade color indicating higher scoring value and the lighter shade color denoting lower scoring value.
